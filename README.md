# Analysis & Design of Algorithmic Paradigms in C++

This repository serves as an advanced computer science archive dedicated to exploring programmatic optimization models, algorithmic time/space complexities, and problem-solving paradigms using object-oriented C++.

## Design Strategies Index

### Phase 1: Complexity Reductions & Subproblems
* **`01_divide_and_conquer.cpp`**
  * Demonstrates spatial fragmentation and recursive combination logic via a clean `O(N log N)` Merge Sort implementation.
* **`02_greedy_method.cpp`**
  * Explores local-ratio optimization tactics using an item-sorting matrix to evaluate Fractional Knapsack problems.
* **`03_dynamic_programming.cpp`**
  * Resolves subproblem duplication bottlenecks via bottom-up matrix tabulation tables to execute exact 0/1 Knapsack constraints.
    ### Phase 2: State-Space Traversals & Tree Pruning
* **`04_backtracking_n_queens.cpp`**
  * Employs non-linear state exploration to resolve combinatorial constraints via systematic step validation and rollback states (N-Queens).
* **`05_branch_and_bound.cpp`**
  * Constructs cost prioritization boundaries combined with bounding algorithms to prune unviable state-space tree vectors efficiently.
    ### Phase 3: Mathematical Reductions & Advanced Optimization Pipelines
* **`06_algebraic_transformation.cpp`**
  * Optimizes linear operational logic spaces into logarithmic computation scales via squaring property transformations.
* **`07_parallel_methods.cpp`**
  * Splits core system calculations across asynchronous concurrent multi-threaded execution tracks using native system utility thread allocations (`std::thread`).
* **`08_network_flow_methods.cpp`**
  * Resolves graph network pipeline capacity constraints utilizing structural Residual Graphs and Ford-Fulkerson bottleneck extraction paths.
